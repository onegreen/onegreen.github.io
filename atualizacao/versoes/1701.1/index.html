Relação de mudanças: v.3.0.1701.1

  • Acesso fácil às mudanças ocorridas no sistema e ao Manual do Usuário
    •  Para visualizar a página de mudanças ocorridas na versão, basta clicar no link existente no número da versão.

  • Melhoria no acesso às legendas do Farol de Empreendimentos


  • Definição das Metas para os Gráficos de indicadores
    •  Para que seja possível alterar as metas, é necessário que a permissão na Manutenção de Segurança seja definida como "Sim" para o Usuário ou Grupo de Usuário desejado.



  • Gráficos de Timeline e Curva S para os cronogramas



  • Possibilidade de definição de valores previstos para os Estudos Ambientais



  • Possibilidade de definição manual ou automática para o custo previsto das licenças



  • Gráficos e Relatórios de Condicionantes contemplando todos tipos de tarefas
    •  Gráficos: Tarefas Dentro do Prazo
    Relatórios: Cumprimento de Tarefas, Evidências de Execução das Tarefas, Reprogramações de Tarefas



Para mais informações, acesse o Manual do Usuário.
