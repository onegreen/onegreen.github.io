Relação de mudanças: v.3.0.1808.5

  • Novo status "Em renovação" para as licenças definido automaticamente
    •  Ao relacionar uma licença à outra, a licença relacionada será definida como "Em renovação" (cor do farol marrom) até que a licença seguinte seja obtida, o que fará com que a licença anterior seja concluída.

  • Status automáticos

    • Os status das licenças são definidos automaticamente de acordo com o avanço das mesmas, podendo serem concluídas, indeferidas ou canceladas a qualquer momento pelo usuário.


  • Cadastro simplificado dos estudos

    • Os estudos ambientais não mais obrigam informar contrato relacionado ao mesmo, podendo ser lançado no sistema para posterior definição do contrato atrelado.

  • Definição manual da data limite de renovação da licença

    • Toda modificação na data limite de renovação é armazenada no sistema informando quem alterou, quando e o porquê.

  • Possibilidade de exportar os dados do gráfico de Status das Licenças para Excel


  • Melhorias na rastreabilidade e pesquisa dos dados das solicitações

    • Agora a listagem das solicitações permite filtro avançado, na edição das mesmas, é possível acessar o projeto/licença relacionada e na licença/projeto é possível acessar os dados da solicitação relacionada.


  • Possibilidade de adicionar logo da empresa nos relatórios

    • Os relatórios PDF agora possibilitam a impressão com a logomarca da empresa, podendo definir o posicionamento da mesma no arquivo. Para ter acesso à esta funcionalidade, a equipe de suporte da Onegreen deverá ser contactada para configuração, sendo que para esta deverá ser enviado o arquivo da logomarca no formato PNG.




Para mais informações, acesse o Manual do Usuário.
